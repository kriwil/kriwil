gentoo: patching source before emerge

Penting nih, baru belajar gentoo soalnya. Patch source code sebelum diinstall.

# ebuild myebuild fetch (if you don't have it in distfiles)
# ebuild myebuild unpack (unpacked to /var/tmp/portage/packagename/something)
# ebuild myebuild compile
# ebuild myebuild install
# ebuild myebuild qmerge

Catatan: myebuild merupakan path ke /usr/portage/blabla/appname/app.ebuild

Reddit Membuka Kodenya

Wah, setelah beberapa waktu yang lalu Facebook membuka (sebagian) kodenya, kali ini reddit melepas kodenya dengan lisensi open source.

Dengan membuka kode sumbernya, gue rasa reddit berada di depan digg. Algoritma peringkat digg sering menjadi pertanyaan banyak penggunanya, karena banyak yang merasa ada beberapa berita yang tidak layak masuk halaman depan, tetapi bisa masuk. Dengan dibukanya kode sumber reddit, bukan tidak mungkin para pengguna digg yang tidak puas beralih ke reddit.

Bisa juga kode tersebut disalahgunakan untuk mengangkat berita yang dimasukkan. Pasti akan banyak usaha ke arah sana :D
