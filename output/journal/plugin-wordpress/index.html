Plugin WordPress

Berhubung gue masih baru dalam dunia WordPress, gue jadi masih belum tau plugin apa aja yang masuk dalam kategori recommended untuk dipakai para pengguna WordPress.

Sampai saat ini, plugin yang gue pakai masih sedikit. Ini daftarnya:

  • Browser Sniff, oleh Priyadi Iman Nurcahyo. Plugin untuk mendeteksi browser dan sistem operasi yang digunakan oleh pengisi komentar.
  • Gravatar, oleh Tom Werner. Plugin untuk menampilkan Gravatar.
  • Hello Dolly, oleh Matt Mullenweg. Plugin resmi pertama WordPress.
  • Yahoo/MSN Style Smileys, oleh Priyadi Iman Nurcahyo. Berfungsi menampilkan emoticon bergaya Yahoo Messenger dan MSN Messenger.
  • Markdown, oleh Michel Fortin. Menggunakan Markdown Syntax.
  • Recent Links, oleh Markku Seguerra. Menampilkan recent-links. Mengumpulkan link-link yang menurut kita menarik.

Kira-kira, plugin apa lagi ya yang menarik? ada saran?
