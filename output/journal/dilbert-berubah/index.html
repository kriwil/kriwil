Dilbert Berubah!

Pantesan 2 hari terakhir Dilbert gak muncul di widget google personalized home page saya. Ternyata Dilbert hadir dengan tampilan baru. Sekarang comic stripnya berwarna tiap hari. Dulu kan cuma seminggu sekali, pas hari Minggu aja (CMIIW). Oiya, sekarang ada animation juga. Belum saya coba sih, error soalnya.

Windows 2000, Windows XP, Windows XP Media Center 2005, Mac OSX 10.3 or Mac OSX 10.4 is required to view this page

WTF. Padahal cuma flash aja. Nanti dicoba ganti user agent lah.

Ada satu fitur yang menarik, namanya Mashups. Jadi kita bisa buat sendiri kata-kata pada bagian punch line. Ternyata banyak yang bisa buat punch line yang lebih lucu dari komik aslinya.

Anyway, kudos to all dilbert team.
