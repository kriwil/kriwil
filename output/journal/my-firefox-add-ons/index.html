My Firefox Add-ons

Tweet
Fri 22 February 2008
By aldi

In journal.

Meneruskan tulisan sebelumnya, saya jadi mau berbagi add-on firefox yang saya gunakan. Sekalian buat catatan saya kalau nanti pakai windows (yang mana firefox menjadi browser utama). Ini daftarnya.

  • Tab History Menyimpan history dari tab parent. Informasi lebih jelas ada di tulisan sebelumnya.
  • Adblock Plus Penyelamat para fakir bandwidth. Kalau emang gak mau diliat, kenapa juga harus ditampilkan.
  • Greasemonkey Walaupun namanya agak, begitulah, add-on ini benar-benar menunjukkan kecanggihan javascript. Penting buat pengalaman lebih dalam berinternet.
  • PDF Download Ini bagus buat yang pakai windows dan menggunakan super bloated adobe reader. Kalau kita menekan link yang berisi file pdf, kita bisa memilih untuk mendownload atau membukanya.
  • Web Developer Ini jelas add-on wajib buat, euh, web developer. Kalau firefox-nya belum ada ini, belum bisa dibilang web developer (halah).
  • Firebug Saya banyak menggunakan ini buat mendebug javascript. Saya malah baru tau kalau bisa debug html dan css juga. Ini juga tool penting buat web developer.
  • Compact Menu 2 Dengan adanya add-on web developer, toolbar firefox saya jadi memakan tempat, dan saya sebagai pecinta simplicity merasa terganggu dengan itu. Add-on ini mengatasi masalah itu dengan membuat menubar menjadi satu icon saja.
  • Fission Menggabungkan progress bar firefox dengan address bar-nya. Ini gayanya safari osx. Keren menurut gue.
  • Stop-or-Reload Button Menggabungkan tombol stop dan reload. Kalau halaman sedang load, tombol berbentuk stop. Kalau sudah selesai load, tombol berbentuk reload. Coba sendiri deh.
